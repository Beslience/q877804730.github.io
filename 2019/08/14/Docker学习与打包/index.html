发表于 2019-08-14

war包（SpringMVC项目）镜像部署

  • 1.拉取tomcat镜像
  • 2.上传需要部署的war包到自定义的工作目录下
  • 3.在上个步骤创建的工作目录下创建Dockerfile文件。将war包和Dockerfile文件放在统一目录下。
  • 4.创建镜像（tip：-t表示为该镜像起名称；构建命令后面有个英文实心点，表示当前目录）
  • 5.通过docker images 查看镜像是否创建成功
  • 6.运行自己的镜像，查看是否正常启动（tip：-d表示在后台运行镜像；-p表示本机的端口对应容器的端口）
  • 7.docker镜像导出，tar镜像可以拷贝到其他机器docker环境中加载
  • 8.docker镜像加载

Docker常用操作

  • 查看所有容器
  • 停止容器运行
  • 删除容器 （-f强制删除，正在运行的容器也删除）
  • 删除镜像（-f强制删除，否则需先停止容器运行才能删除镜像）
  • 镜像打tag（同更新tag）
  • 查看指定镜像
  • 查看指定进程
  • 将镜像push到镜像仓库
  • 进入正在运行的容器执行命令（exec命令进入容器再退出后，容器不会停止；attach进入容器再退出后，容器停止）
  • 退出容器
